Getting GUIDO Server information¶

Path

server

Response body
  • in case of success:
    a version number of the server, formatted as JSON with ‘server’ as key.
    Example: { "server" : "1.0" }
Response code

  • 200 (“Success”)

Example¶

Using jQuery:

$.get('http://guido.server.org/server');
